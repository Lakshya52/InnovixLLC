================================================================================
INNOVIX LLC - COMPLETE PROJECT DOCUMENTATION & MANAGEMENT GUIDE
================================================================================

This document provides a comprehensive overview of the InnovixLLC platform, 
covering the Public Website, User Dashboard, and Admin Dashboard functionalities.

--------------------------------------------------------------------------------
1. PUBLIC WEBSITE (FRONT-END)
--------------------------------------------------------------------------------
The public website is accessible to all visitors and provides information about 
products, services, and the company.

- Home Page (/): 
  - Featured hero section with key value propositions.
  - Showcases featured products and latest blog posts.
  - Quick links to services and categories.

- Products Catalog (/products):
  - Displays all products with "Live" status.
  - Filterable by categories (OS, Productivity, Enterprise, etc.).
  - Search functionality to find specific software.

- Product Details (/productsdetails/[id]):
  - Comprehensive view of a single product.
  - Includes pricing, SKU, features (trust signals), and system requirements.
  - "Add to Cart" functionality for purchasing.

- Blogs & Articles (/blogs):
  - List of all "Published" blog posts.
  - Search and filter by tags/categories.
  - Blog Details (/blogsdetails/[slug]) features high-quality reading experience 
    with "Key Takeaways" highlighted at the top.

- Support & Interaction:
  - Contact Us (/contact): Inquiry form for direct communication.
  - AI Chatbot (/chatbot): Instant support for common queries.
  - Shopping Cart (/cart): Manage items before proceeding to checkout.

- Legal Pages:
  - Privacy Policy (/privacy-policy)
  - Refund Policy (/refund-policy)
  - Terms of Service (/terms)

--------------------------------------------------------------------------------
2. USER DASHBOARD (CLIENT AREA)
--------------------------------------------------------------------------------
Accessible to registered users. Provides tools to manage purchases and support.

- Dashboard Overview (/dashboard):
  - Summary of recent orders and active licenses.
  - Quick access buttons to main features.

- My Orders (/orders):
  - Complete history of all purchases.
  - View order status (Processing, Completed, etc.) and transaction amounts.

- License Keys (/keys):
  - The central hub for digital assets.
  - Users can view and copy their purchased product keys here.
  - Keys are organized by product with clear status indicators.

- Support Tickets (/support):
  - Users can create new support requests.
  - Real-time chat interface to communicate with administrators.
  - Track ticket status (OPEN, IN_PROGRESS, RESOLVED).

- Account Settings (/settings):
  - Profile Management: Update name, email, and upload profile pictures.
  - Security: Change password functionality.
  - Email Preferences: Toggle marketing and transactional email notifications.

--------------------------------------------------------------------------------
3. ADMIN DASHBOARD (CENTRAL MANAGEMENT)
--------------------------------------------------------------------------------
Accessible ONLY to users with the "ADMIN" role at /admin.

A. Dashboard Home (/admin/dashboard):
   - Real-time statistics on sales, user growth, and active tickets.
   - Quick overview of system health.

B. Product Inventory (/admin/inventory):
   - Add/Edit/Delete products.
   - Manage Features (JSON) and System Requirements (JSON).
   - Stock Management: Upload bulk license keys (one per line).
   - Low Stock Alerts: Visual indicators when keys for a product are < 10.
   - System Logs: Tracks all inventory-related changes.

C. User Management (/admin/users):
   - View all registered users.
   - Monitor "Last Active" timestamps to track engagement.
   - Administrative Controls: Block/Unblock users or permanently delete accounts.
   - Role Management: Verify or update user permissions.

D. Order Monitoring (/admin/orders):
   - Global view of all platform transactions.
   - Filter and search orders by user or product.

E. Blog Management (/admin/blogs):
   - Full Markdown editor for creating/editing articles.
   - Manage Tags, "Key Takeaways," and SEO descriptions.
   - Control visibility (Public/Private) and status (Draft/Published).

F. Support Desk (/admin/support):
   - Centralized interface to respond to all user tickets.
   - Real-time messaging with users.
   - Mark tickets as resolved after providing assistance.

--------------------------------------------------------------------------------
4. SYSTEM ARCHITECTURE & TECH STACK
--------------------------------------------------------------------------------
- Framework: Next.js (App Router)
- Database: MySQL with Prisma ORM
- Authentication: NextAuth.js
- Styling: Vanilla CSS & TailwindCSS
- Emails: Nodemailer with Titan SMTP for Transactional and OTP emails.

--------------------------------------------------------------------------------
5. MAINTENANCE & TROUBLESHOOTING
--------------------------------------------------------------------------------
- Database Sync: Use 'npx prisma db push' after schema changes.
- Environment: Ensure all keys (DATABASE_URL, EMAIL_HOST, EMAIL_USER, etc.) are in .env.
- Access Issues: If "Permission Denied," check the 'role' column in the 'User' 
  table for your email.
